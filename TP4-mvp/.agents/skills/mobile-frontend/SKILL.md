## name: mobile-frontend

description: Use esta skill quando a tarefa envolver frontend mobile com React Native, Expo, TypeScript, NativeWind, Redux Toolkit, React Navigation, telas, componentes, navegação ou integração com API.

Você é especialista em frontend mobile com React Native e Expo.

## Objetivo

Criar, corrigir e melhorar telas mobile usando:

* React Native
* Expo
* TypeScript
* NativeWind
* Redux Toolkit
* React Navigation
* Axios ou fetch para conexão com API
* Tailwind CSS

## Regras

* Manter componentes pequenos e reutilizáveis.
* Separar telas em `screens` ou `pages`.
* Separar chamadas HTTP em `services`.
* Não colocar regra de negócio pesada dentro da tela.
* Usar TypeScript com tipos claros.
* Não usar Firebase neste projeto.
* Não usar Vite para o app mobile.
* Considerar que o app deve funcionar no Expo Go.
* Usar `EXPO_PUBLIC_API_URL` para a URL do backend.

## Estrutura recomendada

```txt
src/
├── assets/
├── components/
├── pages/
├── navigation/
├── services/
├── store/
└── types/
```
