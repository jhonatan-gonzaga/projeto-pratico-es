# Documentação das Telas de Funcionalidades

Esta seção detalha as principais telas do aplicativo **Conecta Obras Itacoatiara**, apresentando suas funcionalidades e interfaces visuais. O objetivo é fornecer uma visão clara das interações do usuário e dos recursos disponíveis em cada etapa do sistema.

## 1. Tela de Perfil

A **Tela de Perfil** permite ao usuário visualizar suas informações pessoais e profissionais. É o ponto central para gerenciar dados cadastrais e acessar outras configurações.

![Tela de Perfil](https://github.com/user-attachments/assets/6f6ea13b-2521-496a-9545-549ae47fc473)

## 2. Tela de Alteração do Perfil

Na **Tela de Alteração do Perfil**, o usuário pode editar suas informações cadastrais, como nome, contato, endereço e detalhes profissionais. Esta funcionalidade garante que os dados estejam sempre atualizados.

![Tela de Alteração do Perfil](https://github.com/user-attachments/assets/c59d93b5-9ead-44fa-bf27-4f7bc2964c61)

## 3. Tela Inicial

A **Tela Inicial** serve como o dashboard principal do aplicativo, oferecendo acesso rápido às funcionalidades mais utilizadas e um panorama geral das atividades. É a primeira tela que o usuário vê após o login.

![Tela Inicial](https://github.com/user-attachments/assets/774e5992-0521-4d53-be57-b85ac579c642)

## 4. Tela Buscar por Profissional

A **Tela Buscar por Profissional** permite aos usuários encontrar e filtrar profissionais de diversas áreas, facilitando a contratação de serviços específicos para suas obras.

![Tela Buscar por Profissional](https://github.com/user-attachments/assets/84120cda-e4e5-4ff0-a921-d146595638f0)

## 5. Tela Detalhes do Projeto

A **Tela Detalhes do Projeto** exibe informações completas sobre um projeto específico, incluindo descrição, status, profissionais envolvidos e prazos. Permite um acompanhamento detalhado do andamento da obra.

![Tela Detalhes do Projeto](https://github.com/user-attachments/assets/ac7fdf5c-e8d7-4d8c-8bbd-1e4f6d85700d)

## 6. Tela Configurações

A **Tela Configurações** oferece opções para personalizar a experiência do usuário, gerenciar notificações, privacidade e outras preferências do aplicativo.

![Tela Configurações](https://github.com/user-attachments/assets/4b274349-f552-4585-83bb-dfbd852ac945)

## 7. Tela de Oportunidades (Novos Pedidos)

A **Tela de Oportunidades (Novos Pedidos)** apresenta aos profissionais as solicitações de serviço disponíveis, permitindo que eles visualizem e se candidatem a novos projetos.

![Tela de Oportunidades (Novos Pedidos)](https://github.com/user-attachments/assets/34738300-9e2f-4209-ba3c-e9d5f4eea55c)

## 8. Tela de Oportunidades (Meus Serviços)

A **Tela de Oportunidades (Meus Serviços)** exibe os projetos nos quais o profissional já está engajado ou que foram aceitos, facilitando o acompanhamento de suas atividades correntes.

![Tela de Oportunidades (Meus Serviços)](https://github.com/user-attachments/assets/2a17db7f-0615-4532-9b98-d47ea71ae299)

## 9. Tela Meus Projetos

A **Tela Meus Projetos** lista todos os projetos criados ou gerenciados pelo usuário, oferecendo uma visão consolidada de suas obras.

![Tela Meus Projetos](https://github.com/user-attachments/assets/0848ff71-a0c9-414b-87f3-e4c1f5ed5c88)

## 10. Tela Adicionar Projeto

A **Tela Adicionar Projeto** permite ao usuário criar um novo projeto, inserindo todas as informações necessárias para sua descrição e gerenciamento.

![Tela Adicionar Projeto](https://github.com/user-attachments/assets/cd0add09-fcd9-404b-81ff-8a46ad085c2f)

## 11. Tela Detalhes do Projeto (Profissional)

Esta **Tela Detalhes do Projeto** é voltada para o profissional, exibindo informações relevantes sobre o projeto ao qual ele está vinculado, incluindo tarefas e comunicação.

![Tela Detalhes do Projeto (Profissional)](https://github.com/user-attachments/assets/ffcba270-0677-4b42-8665-da689a91c7bf)

## 12. Tela Editar Projeto

A **Tela Editar Projeto** possibilita a modificação das informações de um projeto existente, garantindo flexibilidade na gestão das obras.

![Tela Editar Projeto](https://github.com/user-attachments/assets/6eeefb22-bae4-47eb-8653-7096a052d4bc)

## 13. Tela Perfil Profissional e Avaliações

A **Tela Perfil Profissional e Avaliações** apresenta o perfil detalhado de um profissional, incluindo suas qualificações, portfólio e avaliações de clientes anteriores.

![Tela Perfil Profissional e Avaliações](https://github.com/user-attachments/assets/25ba1fd9-6789-4182-ae60-757b928bc06d)

## 14. Tela Minha Obra

A **Tela Minha Obra** oferece uma visão específica para o cliente sobre o andamento de sua obra, com atualizações e informações pertinentes.

![Tela Minha Obra](https://github.com/user-attachments/assets/a9e649f7-77c9-4149-88b5-29c4f3090ea6)

## 15. Tela Anunciar

A **Tela Anunciar** permite que os usuários publiquem novas demandas de serviço, descrevendo o projeto e as necessidades para atrair profissionais qualificados.

![Tela Anunciar](https://github.com/user-attachments/assets/4986770c-f23b-4511-9234-0c138ede9f3c)

## 16. Tela Anunciar Candidatos

A **Tela Anunciar Candidatos** exibe a lista de profissionais que se candidataram a um anúncio de serviço, permitindo ao usuário revisar e selecionar o mais adequado.

![Tela Anunciar Candidatos](https://github.com/user-attachments/assets/40f3abf3-dac3-4a6b-85f8-6d07caa003f7)

## 17. Tela Buscar

A **Tela Buscar** oferece uma funcionalidade de pesquisa geral, permitindo ao usuário encontrar informações, projetos ou profissionais dentro do aplicativo.

![Tela Buscar](https://github.com/user-attachments/assets/74d27d34-6c19-4d27-8710-8c7b0d8e8b65)
